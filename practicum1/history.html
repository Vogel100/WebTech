DOM Event Handling

Events were first supported when Netscape 2 was released (1995). There werenâ€™t many events, but they were very powerful, and they became famous quickly (mouseover, mouseout). They could detect when a user would submit a form, and made form-validation possible. Interaction was born.

Event handlers were set inline, as an HTML attribute:

<a href="myLink.html" onclick="alert('Do you wish to leave this page?')">Link</a>

This way of event handling was the standard set by Netscape Navigator 2. All other browsers had to use it, otherwise JavaScript would stop working.
